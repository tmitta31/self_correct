To run this repo you need to start the server by running yarn start in server folder and to start the
client run yart start in the client folder. 
To activate the llm you need to add your groq api key in llm/groqGpt.py file line 39 and set it equal to self.gptKey

To run the object detection run the object_detection.py file 
To run the object segmentation run the yolo.py file
