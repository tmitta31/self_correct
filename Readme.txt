To run this repo you need to run the index.js file in server `node index.js' for the server
 and run the App.js file in client/src `node App.js` to run the client. 
To activate the llm you need to app your groq api key in llm/groqGpt.py file line 39 and set it equal to self.gptKey

To run the object detection run the object_detection.py file 
To run the object segmentation run the yolo.py file
